MAVE Hub!

MacroArchitecture Visual Explorer

Dive into the architecture of a computer, big and small. This project will provide a logical, step-by-step progression from the microscopic behavior of binary data to the macroscopic layout of full computer systems.

Level 0 • The Basics

Macro Architecture Visualizer

See the big picture. Play interactive minigames and watch real-time data buses transfer information across the motherboard between the CPU, RAM, GPU, and specialized hardware.

Hardware, Buses, I/O Launch ➔
Level 1 • The Foundation

Data Representation Lab

Computers don't understand text or decimals; they only understand electrical states. Learn how to convert between Base-2 (Binary) and Base-16 (Hexadecimal), and visualize how real hardware stores complex decimal fractions using the IEEE 754 standard.

Base Math, Floating-Point Launch ➔
Level 2 • Micro-Logic

Logic Gate Sandbox

Build the brain of a computer from scratch. Drag and drop basic logic gates (AND, OR, XOR) to construct complex 8-bit components like Adders, Multiplexers, and D-Type Flip-Flops that grant the system memory.

Boolean Logic, Arithmetic, Memory Launch ➔
Level 3 • Processing

RISC Pipeline Visualizer

Dive inside the CPU core. Write assembly code and watch it execute cycle-by-cycle through the classic 5-stage RISC pipeline (Fetch, Decode, Execute, Memory, Writeback). Understand stalls, data hazards, and branching. Understand why we use RISC instead of CISC.

ISA, Pipelining, Hazards Launch ➔
Documentation

User Manual

The official documentation for the MAVE, contains condensed details of all the topics and terms between the various simulators. Also contains details that could not fit into the tool. Example of content: the manual has a glossary of all supported Assembly operations, detailed breakdowns of pipeline stages, and explanations of system statistics like CPI and Branch Mispredictions.
